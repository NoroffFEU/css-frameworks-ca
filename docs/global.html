formCheck()

A simple form validation. Checks all inputs the validity of all inputs (validated directly in HTML) and creates an array of booleans. Afterwards, the trueChecker function checks if all the booleans are true, and then I extract a variable from this. If all the form inputs are validated in HTML, you're good to go! There is a "pattern" attribute in the email input, which handles regex for me. I could have added this via JS, but hardcoding the regex in HTML did the same job and keeps the code generic and reusable.
