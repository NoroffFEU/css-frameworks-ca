## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:saibia8/css-frameworks-ca.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the project, run the following commands:

```bash
npm run start
```
